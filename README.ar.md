## ما هو؟

**Code Crime Scene** أداة سطح مكتب محلية (ملف تنفيذي واحد) للمطوّرين والفرق اللي يبون قراءة صادقة وسريعة لصحّة قاعدة الكود — **بدون رفع أي سطر مصدري للسحابة**.

توجّه الأداة لمجلّد، فيمشي محرّك Rust على الشجرة، يشغّل التحليل الساكن **بالكامل على جهازك**، وينتج **تقريرًا بأسلوب "Forensic Noir"**: خمس درجات، وحكم مُقدَّر، وقائمة **ملاحظات قابلة للتنفيذ** — كل ملاحظة بدليلها (ملف + سطر + مقتطف سطر واحد)، ومبرّر خطورة رقمي، وخطة إعادة هيكلة آمنة على شكل PRs. المحرّك يقيس **التعقيد الدوراني (cyclomatic) لكل دالة**، ويكشف **الدورات في رسم الاعتمادات**، وينقّب **تاريخ git** عن البؤر الساخنة ومخاطر المالك الوحيد، ويفحص **أكثر من 16 نمطًا من الأسرار المسرّبة** بتحليل entropy.

وفيه رأي ثانٍ **اختياري**: تُرسل خلاصة مضغوطة **بلا أي كود** إلى مزوّد الذكاء اللي تختاره — **DeepSeek أو OpenAI أو Claude (Anthropic) أو نموذج محلّي (Ollama / LM Studio)** — فتحصل على "تقرير المحقّق" مكتوبًا. كودك المصدري ما يطلع من جهازك أبدًا — يُرسل فقط أرقام مجمّعة ومسارات ملفات، وفقط لمّا تطلب أنت.

> **محلّي افتراضيًا. الذكاء بطلبك.** الفحص محلّي 100%، وما يُستدعى أي ذكاء إلا لمّا تضغط **«حلّل بالذكاء»**.

## أبرز المزايا

| | |
|---|---|
| 🔒 **يعمل دون اتصال** | لا backend، لا تسجيل دخول، لا تتبّع. التحليل الساكن يجري محليًا في Rust. |
| 🧮 **تعقيد حقيقي** | تعقيد دوراني (cyclomatic) لكل دالة — وأخطرها تنزل في **«غرفة الاستجواب»** داخل التقرير. |
| 🕰️ **تحقيق في تاريخ git** | البؤر الساخنة، الملفات اللي تتغيّر دايمًا مع بعض، ومخاطر المالك الوحيد (bus factor) — تُنقَّب محليًا من تاريخ مستودعك (**«صحيفة السوابق»**). |
| 🎮 **يفهم Lua و FiveM** | دعم Lua كامل بما فيه موارد FiveM: الأحداث، NUI، الثريدات، الأوامر، أنماط QBCore/ESX/ox، تمييز client/server — وحتى أسرار `server.cfg`. |
| 🧠 **ذكاء اختياري — أنت تختار** | DeepSeek أو OpenAI أو Claude (Anthropic) أو نموذج محلّي Ollama/LM Studio — بطلبك، وبخلاصة بلا كود. |
| 🎯 **أدلّة مرتّبة** | الاستخدام الفعلي يتفوّق على الاستيراد/النوع/التعليق. كل ملاحظة بملف وسطر ومقتطف. |
| 📊 **خمس درجات + حكم** | تقييم المشروع، الدَّين التقني، المعمارية، الخطر الأمني، قابلية الصيانة — مع مؤشرات ▲▼ مقارنةً بفحصك السابق. |
| 🧩 **واعٍ بالسياق** | يميّز الهوك عن نقطة دخول الخادم عن ملف الأيقونات، ويقترح إصلاحًا يناسب الملف. |
| 🛡️ **أسرار v2** | أكثر من 16 نمط مفاتيح (GitHub، Stripe، AWS، Google، Slack، Telegram، JWT، rcon حق FiveM…) + كشف بطريقة Shannon entropy، وكلها تُحجب دائمًا. |
| 🩹 **خطط إعادة هيكلة آمنة** | كل ملاحظة معها تقطيع PR-by-PR + أوامر التحقّق (`npm run`) بالضبط. |
| 🌍 **ثنائي اللغة** | واجهة كاملة إنجليزي/عربي (RTL)، والذكاء يردّ بلغتك. |
| 🔑 **المفاتيح في خزنة النظام** | مفاتيح الـ API في Windows Credential Manager (خانة لكل مزوّد) — لا في ملف، ولا في الواجهة. |
| 🗄️ **سجلّ محلّي** | كل تقرير يُحفظ في قاعدة SQLite محلية تفتحها وقت ما تبي. |

## 🔁 جرّبناها على نفسها — حلّينا الأداة بالأداة

أغلب أدوات الجودة ما تُوجَّه أبدًا على كودها هي. نحنا وجّهنا Code Crime Scene على **كوده نفسه** — فأدانَنا.

أول فحص ذاتي كان قاسيًا: **51 / 100 · تقدير F · «جريمة قتل».** وكان **محقًّا**. سمّى ديونًا حقيقية:

- **12.1% كود مكرّر** في المشروع
- **ملفات عملاقة** تتجاوز 400 سطر بكثير — محرّك التحليل وحده **1,718** سطرًا، ومحرّك الملاحظات **770**، وطبقة الترجمة **889**، وملف أنماط واحد **922**
- **تدفّق مفتاح API مكرّر** منسوخ بين شاشتين
- وبعض ملاحظات المحرّك **نفسه** كانت تحتاج دليلًا أدقّ

فسوّينا الصح: ضغطنا زرّ **«نسخ كـ Prompt»** في التقرير، وأعطينا تقرير الأداة الجنائي عن نفسها لمساعد ذكاء برمجي، وأصلحنا **بالضبط** ما سمّته — قسّمنا كل ملف عملاق إلى وحدات مركّزة تحت 400 سطر، واستخرجنا التدفّقات المشتركة، وشدّينا المحرّك. ثم **أعدنا الفحص بنفس الأداة**:

## كيف يشتغل؟

1. **الفحص** — يمشي على المجلّد ويتجاهل `node_modules` ومخرجات البناء و`.git` وملفات القفل والمولّدة.
2. **التحليل** — مقاييس الأسطر، الدوال الطويلة، **التعقيد الدوراني لكل دالة**، تكرار بنوافذ منزلقة، الاستيرادات غير المستخدمة، رسم الاعتمادات مع **كشف الدورات**، و**16+ نمط أسرار بتحليل entropy** — كله في Rust ومحلّي.
3. **تحقيق git** — لو المجلّد مستودع git، تُنقَّب آخر 1000 commit محليًا عن البؤر الساخنة وأزواج التغيير المتلازم ومخاطر المالك الوحيد. ما هو مستودع؟ التقرير يتجاوز اللوحة ببساطة.
4. **التقرير** — درجات (مع مؤشرات ▲▼ مقارنةً بفحصك السابق) وحكم مُقدَّر وملاحظات مرتّبة بأدلّتها وخطط إصلاحها.
5. **(اختياري) حلّل بالذكاء** — خلاصة مضغوطة (أرقام ومسارات، **لا كود**) تُرسل للمزوّد اللي اخترته لتقرير مكتوب.

## التثبيت

**تحميل (ويندوز):** نزّل **[Code-Crime-Scene.exe](https://github.com/ABUGIZA/code-crime-scene/raw/main/Code-Crime-Scene.exe)** (≈11 ميجا) مباشرةً من المستودع وافتحه بنقرتين — ملف واحد مستقلّ، بلا مثبّت وبلا اعتماديات.

> ⚠️ غير موقّع بعد، فقد يظهر تحذير SmartScreen أول مرّة — اضغط **More info ← Run anyway**.

**ابنِ من المصدر:**
```bash
# المتطلبات: Node 18+، Rust (stable)، ومتطلبات Tauri لنظامك
git clone https://github.com/ABUGIZA/code-crime-scene.git
cd code-crime-scene
npm install
npm run tauri dev      # تشغيل تطويري
npm run tauri build    # إنتاج نسخة مستقلّة
```

## إعداد الذكاء (اختياري)

الذكاء **اختياري** — التقرير المحلي يشتغل كامل بدون مفتاح. أربعة مزوّدين مدمجين:

| المزوّد | المفتاح من | ملاحظات |
|---|---|---|
| **DeepSeek** | [platform.deepseek.com](https://platform.deepseek.com/) | الافتراضي، الأرخص |
| **OpenAI** | [platform.openai.com](https://platform.openai.com/) | |
| **Claude (Anthropic)** | [console.anthropic.com](https://console.anthropic.com/) | |
| **محلّي / مخصّص** | — بلا مفتاح | أي سيرفر متوافق مع OpenAI: Ollama، LM Studio، vLLM… |

1. افتح **الإعدادات ← المحقّق الذكي** واختر مزوّدك (أو من شاشة البداية).
2. الصق المفتاح واضغط **تحقّق واحفظ**. التطبيق يرسل طلبًا حقيقيًا صغيرًا ليتأكّد أن المفتاح يعمل، ثم يحفظه في **خزنة النظام** (خانة لكل مزوّد). للمزوّد **المحلّي/المخصّص** وجّهه فقط لرابط سيرفرك (مثل `http://localhost:11434/v1`) — بلا أي مفتاح.
3. عدّل **النموذج** لكل مزوّد لو حبيت.
4. افتح أي تقرير ← انزل إلى **تقرير المحقّق** ← اضغط **حلّل بالذكاء**.

**ضمانات**
- زرّ **«حلّل بالذكاء»** مخفي حتى تربط مزوّدًا، والـ backend يرفض التشغيل بدونه. لا مفتاح ⇐ لا استدعاء للذكاء، أبدًا (السيرفرات المحلية استثناء، وبقرارك الصريح).
- **كودك المصدري ما يُرسَل أبدًا**. تطلع أرقام مجمّعة ومسارات فقط، وفقط عند تلك الضغطة.
- المفاتيح تُقرأ **في Rust فقط** عند استدعاء المزوّد، وما تلمس طبقة الويب.

## إضافة موزّع/خدمة ذكاء أخرى (مفتوح المصدر 💚)

كل تكامل الذكاء في **ملف واحد**: [`src-tauri/src/ai.rs`](src-tauri/src/ai.rs)، يضمّ أربعة مزوّدين خلف دالتين:

```rust
pub async fn verify_key(provider: &str, base_url: &str, key: &str) -> Result<(), String>;
pub async fn analyze(provider: &str, base_url: &str, key: &str,
                     model: &str, summary: &str, lang: &str) -> Result<String, String>;
```

أغلب الخدمات المتوافقة مع OpenAI تشتغل اليوم مباشرة عبر خيار **محلّي/مخصّص** (رابط فقط). ولإضافة مزوّد من الدرجة الأولى (Gemini، Mistral، …): أضف فرعًا في الـ match، وأبقِ نفس **برومبت النظام** (`SYSTEM_PROMPT` / `SYSTEM_PROMPT_AR`)، وحوّل الردّ إلى Markdown، وسجّله في قائمة `AI_PROVIDERS` في [`src/lib/types.ts`](src/lib/types.ts). الواجهة وتخزين المفاتيح واللغة وعرض التقرير تبقى كما هي. الـ PRs مرحّب بها.

## الخصوصية والأمان

- **لا يطلع أي كود مصدري من جهازك.** استدعاء الذكاء الاختياري يرسل أرقامًا مجمّعة ومسارات فقط.
- **لا backend، لا تسجيل دخول، لا تحليلات.** التطبيق ما يعمل أي اتصال شبكي إلا لمّا تضغط «حلّل بالذكاء».
- **المفاتيح في خزنة النظام** (Windows Credential Manager، خانة لكل مزوّد) — لا تُكتب على القرص كنصّ، ولا تُكشف للواجهة.
- **التقارير محلية** في SQLite داخل مجلّد بيانات التطبيق.
- كشف الأسرار محاولة ساكنة مبدئية — وهو **ليس** تدقيقًا أمنيًا.
